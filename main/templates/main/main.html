Available Api's


.../api/myaccount :

		Authentication:
		Token authentication
		GET :
			output : datail of user such as username, email, ...
				and incomes and expense in last month

.../api/expense : 

		change user's wallet.

		Authetication:
			Token authentication

		POST :
			input : text, value, date(optional)
			output : this expense, status=201

		GET :
			input : text(optional), value(optional),
				from(optional, defualt=last month), to(optional, defualt=now)
			output : current user's expenses

.../api/income : 

		change user's wallet.

		Authetication:
			Token authentication

		POST :
			input : text, value, date(optional)
			output : this income, status=201

		GET :
			input : text(optional), value(optional),
				from(optional, defualt=last month), to(optional, defualt=now)
			output : current user's incomes
.../api/register : 

		register user

		POST :
			input : username, password1, password2
			output : username, password, token
